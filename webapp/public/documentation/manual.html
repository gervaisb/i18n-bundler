What does this app ?

The goal is to help managing messages bundle. This app provide a simple frontend who allow users to manage a set of translated messages and expose is backend trough a RESTfull API.

How to use it ?

This application is built around 3 concepts :

Translation
A translation is a set of messages in different languages. Thoses messages are identified a key.
Bundle
A bundle is a set of Translations. It allow users to regroup the translations. (eg: messages, errors)
Project
The project is the base unit, it contains one or more Bundles.

Project

To create a project, open the home page and use the "New" link from the header navigation.

A project is identified by a groupId and and artifactId.

Bundle

When you are viewing a project, you can create a bundle to this project whith the "New" link from the header navigation or via the "+" tab.

A bundle is identified by a name who must be unique inside a project.

Translation

A translation is key and and some pairs of language and message. You can manage translations inside a bundle and export them a Java .properties or Microsoft Excel sheet.

Note : Actually only French and Dutch are supported as language.

Add a translation

When you edit a project you have a two columns page. The left column contains all bundles for the project inside tabs while the left column let you add translation in the selected bundle.

The translation is identified by a key who is unique inside the project. She can also contains values in some languages. If you leaves those values empty, you can let the system translate the key for you.

The automatic translation is done on a humanized form of the key who is supposed to be in english.

Editing a translation

Place your cursor on the translation row, a cross and a pencil appears on the left. Use the pencil to open a dialog who let you edit the translation key and values.

Since a translation key is unique inside a bundle, you can also edit a translation by creating a new one with the same key, you then override the existing translation.

Remove a translation

To remove a translation, use the cross who appear on the left when you move your cursor hover the translation row.
